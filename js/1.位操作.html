# 位操作

一补数（反码）：按位非，每个0变成1，每个1变成0 二补数（补码）：确定绝对值的二进制表示，取二补数（反码），再结果+1

# 负值

负值 = 二补数

如18，二进制为 0000 0000 0000 0000 0000 0000 0001 0010

计算一补数（反码）为 1111 1111 1111 1111 1111 1111 1110 1101

结果+1为 1111 1111 1111 1111 1111 1111 1110 1110

# 位操作符

  • 1.按位非（~）

返回一补数（反码），最终结果为对数值取反并减1

let num1 = 25; // 二进制 00000000000000000000000000011001
let num2 = ~num1; // 二进制 11111111111111111111111111100110
console.log(num2); // -26
  • 2.按位与（&）
let result = 25 & 3;
console.log(result); // 1
25 0000 0000 0000 0000 0000 0000 0001 1001
3 0000 0000 0000 0000 0000 0000 0000 0011
AND 0000 0000 0000 0000 0000 0000 0000 0001
  • 3.按位或（|）
let result = 25 | 3;
console.log(result); // 27
25 0000 0000 0000 0000 0000 0000 0001 1001
3 0000 0000 0000 0000 0000 0000 0000 0011
AND 0000 0000 0000 0000 0000 0000 0001 1011
  • 4.按位异或（^）

只在一位上是 1 的时候返回 1（两位都是 1 或 0，则返回 0）。

let result = 25 ^ 3;
console.log(result); // 26
25 0000 0000 0000 0000 0000 0000 0001 1001
3 0000 0000 0000 0000 0000 0000 0000 0011
AND 0000 0000 0000 0000 0000 0000 0001 1010
  • 5.左移（<<）
let oldValue = 2; // 等于二进制 10
let newValue = oldValue << 5; // 等于二进制 1000000，即十进制 64
  • 6.有符号右移（>>）

移位后就会出现空位。右移后空位会出现在左侧，且在符号位之后。 ECMAScript 会用符号位的值来填充这些空位

let oldValue = 2; // 等于二进制 10
let newValue = oldValue << 5; // 等于二进制 1000000，即十进制 64
  • 7.无符号右移（>>>）

无符号右移操作符将负数的二进制（正数的二补数）表示当成正数的二进制表示来处理

let oldValue = -64; // 等于二进制 11111111111111111111111111000000
let newValue = oldValue >>> 5; // 等于十进制 134217726
