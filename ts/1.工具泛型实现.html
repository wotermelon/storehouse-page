# 工具泛型实现

  • keyof 可以用来取得一个对象接口的所有 key 值.
interface Foo {
  name: string;
  age: number
}
type T = keyof Foo // -> "name" | "age"
  • in 可以遍历枚举类型
type Keys = "a" | "b"
type Obj =  {
  [p in Keys]: any
}
  • extends 表示类型的子集

对于联合类型来说会自动分发条件，例如 T extends U ? X : Y, T 可能是 A | B 的联合类型, 那就变成(A extends U ? X : Y) | (B extends U ? X : Y)

// 如果泛型参数 T 为 null 或 undefined，那么取 never，否则直接返回T。
type NonNullable<T> = T extends null | undefined ? never : T

let a: NonNullable<number> // => number
let b: NonNullable<string> // => string
let c: NonNullable<undefined | null> // => never
  • infer 推断一个类型变量
// 如果泛型变量T是 () => infer R的子集，那么返回通过infer获取到的函数返回值，否则返回boolean类型
type Func<T> = T extends (() => infer R) ? R : boolean

// number不是() => infer R的子集，=> boolean
let func1: Func<number>
// () => Promise<number> 是 () => infer R的子集，=> Promise<number>
let func2: Func<() => Promise<number>>

# Partial<T>

Partial 作用是将传入的属性变为可选项

keyof T 拿到 T 所有属性名, 然后 in 进行遍历, 将值赋给 P, 最后 T[P] 取得相应属性的 ? 可选值

实现：

type Partial<T> = {
  [K in keyof T]?: T[K]
}

使用：

interface Person = {
  name: string,
  age: number
}

type PartialPerson = Partial<Person>

// type PartialPerson = {
//   name?: string,
//   age?: number
// }

# Required<T>

Required 作用是将传入的属性变为必选项

-? 是将可选项代表的 ? 去掉

实现：

type Required<T> = {
  [K in keyof T]-?: T[K]
}

使用：

interface Person = {
  name?: string,
  age?: number
}

type RequiredPerson = Required<Person>

// type RequiredPerson = {
//   name: string,
//   age: number
// }

# Readonly

Readonly 将传入的属性变为只读选项

实现：

type Readonly<T> = {
  readonly [K in keyof T]: T[K]
}

使用：

interface Person = {
  name: string,
  age: number
}

type ReadonlyPerson = Readonly<Person>

// type ReadonlyPerson = {
//   readonly name: string,
//   readonly age: number
// }

# Record<K, T>

Record 将 K 中所有的属性的值转化为 T 类型

实现：

type Record<K extends keyof any, T> = {
  [P in K]: T
}

使用：

interface Person = {
  name: string,
  age: number
}

type RecordPerson = Record<'aa'|'bb', Person>

// type RecordPerson = {
//   aa: Person,
//   bb: Person
// }

# Pick<T, K>

Pick 从 T 中取出 一系列 K 的属性

实现：

type Pick<T, K extends keyof T> = {
  [K in T]: T[K]
}

使用：

interface Person = {
  name: string,
  age: number,
  other: any
}

type PickName = Pick<Person, 'name' | 'other'>

// type PickName = {
//   name: string,
//   other: any
// }

# Exclude<T, U>

Exclude 作用是从 T 中找出 U 中没有的元素

实现：

type Exclude<T, U> = T extends U ? never : T

使用：

// (1 extends 1 | 3 ? never : 1) | (2 extends 1 | 3 ? never : 2)
type T = Exclude<1 | 2, 1 | 3>

// type T = 2

# Omit<T, K>

Omit 作用是从 T 中删除对应的 K

实现：

type Omit<T, K extends keyof T> = Pick<T, Exclude<keyof T, K>>

使用：

interface Person = {
  name: string,
  age: number
}

type OmitAge = Omit<Person, 'age'>

// type OmitAge = {
//   name: string
// }

# Parameters<T>

Parameters 当传入的指定类型为函数类型时，返回函数入参的类型

实现：

type Parameters<T extends ((...args: any) => any)> = T extends ((...args: infer P) => any) ? P : never

使用：

// 定义一个函数类型 GetUserFunc，入参 name 的类型是 string
interface GetUserFunc {
  (name: string): any
}

// 提取到了 GetUserFunc 的参数类型
type funcParamsTypes = Parameters<GetUserFunc>
// type funcParamsTypes = [string]

# ReturnType<T>

ReturnType 获取函数返回值的类型

实现：

type ReturnType<T> = T extends ((...args: any[]) => infer R) ? R : any

使用：

function foo(x: number): Array<number> {
  return [x];
}

type FnRt = ReturnType<typeof foo>;

// type FnRt = number[]

# Extract<T, U>

Extract 取出两个类型集合的交集

实现：

type Extract<T, U> = T extends U ? T : never

使用：

type T = Exclude<1 | 2, 1 | 3> // -> 1
